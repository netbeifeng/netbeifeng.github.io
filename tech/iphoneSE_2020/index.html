此时发现我们已经能够成功滚动，但是出现了3个问题。

  1. 当滚动条处于最上方时，芯片的高和宽等于0。
    这个比较好解决，直接在 CSS 那边 给 the-chip 加上 min-width 和 min-height 都为 100 px 就可以了。
  2. 当滚动条处于最下方时，芯片还是不够大。
    这个也很好解决，直接把放大倍率改大一点就可以了，所以将上面代码的10倍，改成20倍。
  3. 在刚开始滚动的时候放大比较快，然而在快滚到底部时比较慢。
    这其实是一个加速度问题，可以参考Easings.net上的easeInCubic曲线。 可以在 Math Function 看到直接将参数乘以3倍就能得到我们需要的曲线。
